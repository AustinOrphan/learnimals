Variable-Based Design System

Our new variable system makes theming more powerful and consistent:

  • All colors are defined as CSS custom properties in a logical hierarchy
  • Base colors are defined once and referenced in both light and dark modes
  • Text and background colors swap between modes without redefining palettes
  • Primary and secondary variants for text and background colors
  • Secondary text in light mode becomes primary text in dark mode (and vice versa)
  • Common UI elements share standardized color roles (primary, secondary, accent)
  • Theme-specific colors are used in preference to generic grays
  • Adding new themes only requires defining the base color set

Color Variable System Benefits

Our new color variable system offers significant improvements:

  • Shared base color palette between light and dark modes
  • Theme-specific colors used in preference to generic neutrals
  • Primary and secondary text/background colors that swap between modes
  • Each theme defines its own set of 8 base colors
  • Light and dark modes use related colors from the same palette
  • Easy modification of entire themes by changing just base colors
  • Consistent naming pattern for all themes (color-1 through color-6, light, dark)
  • More natural color relationships between light and dark modes
